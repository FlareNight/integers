Introduction to Integers

Definition

The numbers..., -3, -2, -1, 0, 1, 2, 3,... together are called integers and their set is denoted by Z. Integers are whole numbers which can be positive, negative, or zero.

Types of Integers

Positive Integers

+1, +2, +3,... are called positive numbers/integers. They are generally written as 1, 2, 3,...

Negative Integers

-1, -2, -3,... are called negative numbers/integers.

Zero

Zero (0) is neither positive nor negative.

Number Line

A Number line of Integers:

  • Has Zero in the middle
  • Has positive integers to the right of 0
  • Has negative integers to the left of 0

Absolute Value

The absolute value of an integer is its numerical value irrespective of its sign.

The absolute value of a positive number is the number itself.
The absolute value of 0 is 0 itself.
The absolute value of a negative integer is its opposite, that is a positive number.
Thus, the absolute value of a number is never negative.
The symbol | | is used for representing absolute value.

Operations on Integers

Properties

Addition

If a and b are two integers, then a + b = c. Ex: -4 + 7 = +3. Holds Good

Subtraction

If a and b are two integers, then a - b = c. Ex: -4 - 7 = -11. Holds Good

Addition

If a and b are two integers, then a + b = b + a. Ex: -4 + 7 = 7 + (-4) = +3. Holds Good

Subtraction

a - b ≠ b - a. Ex: -4 - 7 ≠ 7 - (-4). -11 ≠ +3. Doesn't Hold Good

Addition

a + (b + c) = (a + b) + c. Ex: -3 + [(-5) + 6] = [(-3) + (-5)] + 6 = -2 = -2. Holds Good

Subtraction

a - (b - c) ≠ (a - b) - c. Ex: -3 - [(-5) – 6] ≠ [(-3) - (-5)] – 6. +8 ≠ -4. Doesn't Hold Good

Addition

a + 0 = 0 + a = a. Ex: -5 + 0 = 0 + (-5) = -5. 0 is the additive identity element.

Subtraction

a - 0 = a but 0 - a ≠ a. Ex: –5 – 0 = −5; 0 – (-5) = +5. There is no identity element.

Addition Rules

The sum of two positive integers is the sum of their absolute values with a positive sign.
Ex: (+3) + (+2) = +5
The sum of two negative integers is the sum of their absolute values with a negative sign.
Ex: (-3) + (-2) = -5
The sum of two integers having opposite signs is the difference of their absolute values with the sign of the integer of the greatest absolute value.
Ex: (-3) + (+2) = (-1); (+3) + (-2) = (+1)

Subtraction

Subtracting an integer is the same as adding its additive inverse.

If a and b are two integers, then a - b = a + (-b)

Example: -3 – 5 = -3 + (-5) = -8

Additive Inverse

If a is an integer, then a + (-a) = 0. Here, a and -a are called additive inverses of each other.

Example: (-6) + (+6) = 0
